# 地图开发中常用到的数据

主要包括各省市县的边界线数据,和部分乡镇边界线数据,POI 数据和建筑白模,高程数据,倾斜摄影数据,以及地图瓦片数据数据等。数据主要平时练习使用,请勿于商业活动.数据收集网络,不保证标准和时效性。
